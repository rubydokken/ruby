# Personal Website Starter

This is a static GitHub Pages test site.

## Files

- `index.html` is the page GitHub Pages will serve first.
- `styles.css` controls the visual styling.
- `.nojekyll` tells GitHub Pages to publish the files exactly as-is.

## Publish With GitHub Pages

1. Create a new public GitHub repository.
2. Upload these files to the repository root.
3. Open the repository settings.
4. Go to **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Choose the `main` branch and `/root`, then save.
7. Wait a minute or two for GitHub to show the live site URL.

For a user site, name the repository `YOUR-GITHUB-USERNAME.github.io`.
For a project site, any repository name works and the site URL will include that
repository name.
