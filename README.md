# Ruby Dokken Portfolio Site

This is a one-page personal portfolio site for Ruby Dokken, a freshman at
Rhode Island School of Design.

## Site Overview

- The site opens with Ruby's name, school line, email, and website link.
- Below the header is a vertical image gallery.
- Each image appears on the same page with a simple caption.
- Captions are based on the image filenames, with numbers and file extensions
  removed.
- The site is built as plain static HTML and CSS for GitHub Pages.

## File Structure

- `index.html` contains the page content and image list.
- `styles.css` controls layout, typography, spacing, and colors.
- `Images/` contains the portfolio images shown on the page.
- `CNAME` connects the site to `rubydokken.com`.
- `.nojekyll` tells GitHub Pages to publish the files exactly as-is.
